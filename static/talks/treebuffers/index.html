Real-Time Tree Buffer

Operations:

Legend:

  • scheduled for deletion
  • numbers are distances from root

Theorem

Consider a sequence of $\ell$ modifying operations. Every operation takes $O(1)$ time in the real-time algorithm. We have $s_{\sf rt}^{(k)} \in O(\max_{j\le k} s_{\sf gc}^{(j)})$ for all $k \le \ell$. If the sequence is extensive then $s_{\sf rt}^{(k)} \in O(s_{\sf gc}^{(k)})$ for all $k \le \ell$.

Example Applications

  • runtime verification: error traces
  • regex searching: capturing groups
  • tracing infinite automata with relevant/irrelevant transitions
