Tentatively useful

FreeBoogie. It aims to be a backend for verification tools such as Spec# and ESC/Java2. At the moment it is just a frontend for the Boogie Programming Language (BoogiePL) designed by the Spec# team. A frontend for Spec# (the programming language) is available from Microsoft. A frontend for Java+JML is available from ETH Zurich.

CFIND. A dead project which I hope to revive some day. It is supposed to offer fast text search across all files on UNIX systems. The `alpha' (and only) version indexes only .txt files and is way too memory hungry. (My not-quite-working local copy has some improvements to index html, tex, and pdf.)

SPOJ

Hit or miss. A slightly tricky simulation. (noweb, pdf, cpp)

Selfish cities. The Hungarian algorithm. (noweb, pdf, cpp)

Landscaping. Dynamic programming on a tree. Has some interesting complexity analysis. (cweb, metapost, pdf, cpp)

Assembler circuits. Hash-consing. (cweb, pdf, cpp)

Others

A cute data structure, the min-queue. (cweb, pdf, cpp)

A program that reads in a list of Debian package descriptions and dumps a DIMACS SAT CNF format, which is supposed to unambiguously express the semantics. This hack plus a 10-lines hack in MiniSAT do the same as edos-debcheck, modulo some bugs. (python html)

My road to solving problem G from NWERC2007 (ACM ICPC regional). The task is to efficiently find summits given a height map. (bz2 archive with three programs, from slow to fast)

`In-place' inverse of a permutation (Algorithm J and exercise 1.3.3-13). (cweb, pdf, c)

Multiplication of cycle permutations (exercise 1.3.3-8). (cweb, pdf, c)

`In-place' inverse of a permutation (Algorithm I). (cweb, pdf, c)

This is inspired by Knuth's article Mathematics and Computer Science: Coping with Finiteness. The program counts self-avoiding walks on a rectangular grid by brute force. Much smarter techniques exist. (cweb, pdf, c)

Counting some strange strings. Actually I encountered these strings while thinking about how to optimally transform a directed acyclic control flow graph into dynamic single assignment form. (cweb, pdf, c)

A simple SAT solver. (cweb, pdf, c)
