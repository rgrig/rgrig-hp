Research

jStar is a Java program verifier that uses separation logic for annotations. I am just starting to get familiar with the code and I'll be contributing in the coming years.

FreeBoogie is a verification condition generator for the Boogie language. It is the subject of my (submitted) PhD thesis.

ESC/Java is a static verifier for Java programs annotated with JML. I implemented a big part of the reachability analysis that is activated by the -era flag.

CLOPS is a domain specific language, a tool, and a library that make it easy to write Java programs with a command line interface. I contributed to its design.

Fx7 is an SMT solver. I wrote a small module for simplifying formulas in certain circumstances and I might maintain it a bit in the future.

Utilities

CFIND is text file indexer and searcher. It is severely out of date, but I still hope to find time to improve it.

TwitStat finds trending topics and websites from the twits in a certain area by using PageRank. Was developed for fun but might be upgraded to "research" some day.

HomeworkEval is a web application I used to quickly grade programming homeworks.

Javadoc Reader is a web application that makes it easier (for me) to navigate the Java API.

TexPar is a tool I use to (re)format (parts) of my TeX files.
